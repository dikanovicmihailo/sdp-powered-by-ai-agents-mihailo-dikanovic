# Software Development Processes Powered by AI Agents

A hands-on course where you build a suite of AI agents that automate core software development workflows — from git operations and architecture design to CI/CD pipelines and test-driven development. Each module delivers a working agent you can use in real projects.

## What You'll Build

- A **Git workflow agent** that automates branching, commits, and pull requests
- A **Software Architecture agent** that generates arc42 docs and C4 diagrams
- A **Requirements agent** that derives user stories using DDD and Pareto prioritization
- A **CI/CD agent** that scaffolds GitHub Actions pipelines and AWS SAM deployments
- A **TDD/BDD multi-agent system** with specialized test writer, executor, implementer, and refactorer agents
- A **Subscription Platform** built end-to-end using all agents from previous modules
